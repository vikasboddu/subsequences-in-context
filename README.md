# subsequences-in-context

'brew install node' will install Node.JS on Macs with Homebrew (https://nodejs.org/en/download/)
'npm install' will download the dependencies and testing framework 
'npm test' will run the tests
'npm start' will run the server

./demo/post.js can be modified to run any use case that you want
check out https://www.npmjs.com/package/request for the documentation
